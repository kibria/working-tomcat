Welcome to Content Studio

Escenic Content Studio is the content creation, editing and management application of the Escenic content management system. It is the system's "front end", the most visible part of the system, and is intended to be used on a daily basis by the writers and editors responsible for the content of an Escenic-managed web site.

If you're new to Content Studio, it's probably a good idea to browse through the User Guide before you start. If you're too impatient for that, then you could try to:

  • Create a new content item by clicking File > New in the menu bar, or
  • Display the Search panel by clicking on the Search button at the top of the left side of the window. You can use this panel to search for existing content items and open one for editing.


New in version 5.7

Multi-device preview

Content Studio has a completely new, improved preview function for content items and section pages. Previews are now displayed on a tab in the section page/content item editor, and you can switch between large, small and medium options to see how the page looks on different devices. The large, small and medium options can each be set to a range of different resolutions, allowing you to preview many different device sizes. The new preview function also has a time control which lets you preview a page's appearance at different times in the future, so you can see the effects of planned publish/expiry events. The preview tabs can be undocked into a separate window.

Moving sections

You can now use Content Studio to move sections to different locations in the section tree and move all of a section's content items to a new section.

Pinning list items

You can now pin list items in position, so that they do not move up or down when other list items and added and removed.

Performance monitor

A new performance monitor in the Content Studio status bar will alert you if the Content Engine's performance drops below specified levels.

Dockable related content area

Content editors' related content areas are now dockable.

And for developers...

It's now possible to extend Content Studio's search function so that it can search in any external data sources that provide an OpenSearch interface. Such external searches can be fully integrated with the standard search functionality.
